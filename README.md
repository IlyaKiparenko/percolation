# Базовая реализация
Шаблон репозитория, содержащего тесты Travis Ci и AppVeyor.

## Особенности
1. Все проекты должны располагаться в директории **solutions** и иметь расширение **vcxproj**.
2. После сборки проектов выполняется запуск всех найденных исполняемых модулей из директории **Release**.

## Использование
1. Выполнить fork репозитория.
2. Зарегистироваться на https://travis-ci.org/ и https://www.appveyor.com/ с использованием аккаунта GitHub.
3. Добавить текущий репозитории в Travis Ci (Add new repository) и AppVeyor (New project).
4. Выполнить в ваш репозиторий Push или Pull request.
